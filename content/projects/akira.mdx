A modern SaaS starter kit that handles the boilerplate so you can ship product features faster. Auth, payments, database, and transactional email all wired up out of the box.

Authentication via Better Auth covers Google, GitHub, and email sign-in. Payments are powered by Dodo, with Resend and React Email handling transactional flows. Drizzle ORM connects to Neon's serverless Postgres for type-safe data access.

Built on Next.js with Tailwind CSS 4 and shadcn/ui for the interface, Motion for animations, and Bun as the runtime. Skip the scaffolding and start building.
